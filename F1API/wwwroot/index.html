Introduction

Welcome to the Formula 1 API, your gateway to all things Formula 1 racing. This documentation is here to guide you through the available resources and how to access them using HTTP requests. Start by checking out the "Getting Started" section to get a better understanding of what is availalbe!

Getting Started

Lets make your first API request to the Formula 1 API!

Base URL

The base URL is the root for all of the API, if you every make a request and you get back a 404 NOT FOUND, you should start by checking the base URL .

http://localhost:5292/api

Get all drivers

You can access all the drivers from the api by using /drivers after the base url.

http://localhost:5292/api/drivers

The response will look like this:

[{"id":1,"name":"Max Verstappen","age":26,"nationality":"Netherlands","imgUrl":"driver-verstappen.jpg","skill":90,"aggression":80,"experience":80},{"id":2,"name":"Lewis Hamilton","age":38,"nationality":"United Kingdom","imgUrl":"driver-hamilton.jpg","skill":90,"aggression":85,"experience":75}...]

Get specific driver by ID

You can access a specific driver by id using /drivers/:id after the base url.

http://localhost:5292/api/drivers/1

The response will look like this:

{"id":1,"name":"Max Verstappen","age":26,"nationality":"Netherlands","imgUrl":"driver-verstappen.jpg","skill":90,"aggression":80,"experience":80}

Get specific driver by name

You can access a specific driver by name by using /drivers/Get?name=Nameafter the base URL

http://localhost:5292/api/Drivers/Get?name=Lewis

It's important to note that you don't need the full name of the driver, however it is case sensistive.

The response should look like this:

[{"id":2,"name":"Lewis Hamilton","age":38,"nationality":"United Kingdom","imgUrl":"driver-hamilton.jpg","skill":90,"aggression":85,"experience":75}]
